# Blue Steel

Blue Steel is a Maya facial-shape editing tool focused on fast iteration for
primary, inbetween, combo, and combo-inbetween blendshape workflows.

It provides:

- A model/view editor UI for browsing and editing large shape sets.
- Name-driven shape logic (primary/inbetween/combo typing from shape names).
- Scene tracker integration to keep UI state synchronized with Maya changes.
- Conversion helpers (for example Simplex import) and utility operations.
